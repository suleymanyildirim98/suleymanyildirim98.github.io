# suleymanyildirim98.github.io

Personal portfolio website for Suleyman Yildirim, AI Research Engineer.

## Preview locally

The site is static and has no build step. Open `index.html` directly or run a simple local web server:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deploy

GitHub Pages publishes the root of the `main` branch for this repository. Push changes to `main` to deploy the site.

## Extend

- Add portfolio case studies as separate pages and link them from a future projects section.
- Update publication cards in `index.html` as new work is released.
- Add analytics or a custom domain later without changing the static site structure.
- Replace the code-native hero graphic with a portrait or project visual if desired.
